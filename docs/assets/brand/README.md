# Kubecode brand assets

The Kubecode mark combines a project rail, split workspace panes, and an active
Session node in a geometric `K`.

## Palette

| Role | Value |
| --- | --- |
| Cobalt | `#5B5CE2` |
| Violet | `#8B5CF6` |
| Dark surface | `#111318` |
| Light surface | `#F7F8FC` |

## Files

- `public/logo.svg` is the canonical color mark.
- `public/favicon.svg` is the browser favicon.
- `kubecode-mark-light.svg` is the monochrome mark for light backgrounds.
- `kubecode-mark-dark.svg` is the monochrome mark for dark backgrounds.
- `kubecode-mark-512.png` is the square raster export.
- `kubecode-social-preview.svg` is the editable social-card source.
- `kubecode-social-preview.png` is the GitHub social-preview export.

## Usage

- Preserve the mark's aspect ratio and transparent padding.
- Use the color mark or a monochrome variant without recoloring individual
  segments.
- Do not place Agent logos inside the Kubecode mark.
- Do not add shadows, outlines, rotation, or text inside the mark.
- Use the written name **Kubecode** next to the mark when context is ambiguous.

The SVG sources are deterministic repository assets. Regenerate PNG exports
from their SVG source rather than editing the raster files.
